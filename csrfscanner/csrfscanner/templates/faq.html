Frequently Asked Questions

What is this platform?

This platform is a secure code review tool to detect vulnerabilities and potential threats in your code.

How do I scan my code?

You can upload your code file through the "Scan Code" section. The platform will check for vulnerabilities like SQL injections.

What languages does this tool support?

We currently support Python, Java, PHP, JavaScript, and more. Check the documentation for a full list.

How do I submit feedback?

You can provide feedback on the platform's functionality by visiting the "Feedback" section on your dashboard.
